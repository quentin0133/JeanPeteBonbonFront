# Jean-Petebonbon - Frontend

Bienvenue dans le frontend de **Jean-Petebonbon**, l'interface utilisateur pour interagir avec le bot Discord. Ce projet utilise **Angular 15** pour la gestion de la base de données et l'interaction avec le backend.

## Table des matières

## À propos du projet

Le frontend de **Jean-Petebonbon** permet aux utilisateurs de gérer les comportements du bot discord et de ses rappels, il y a un système de connexion dépendant de comment a été configuré l'API, vous pourrez interagir avec le bot.
Dépendant de l'API

## Technologies

- **TypeScript**
- **HTML/CSS**
- **Tailwind CSS** et **Preline**
- **RxJS**
